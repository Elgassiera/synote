synote.js
=========

Node.js version of Synote

You need to install some npm -g

* mocha
* yo
* sails
* generator-angular
* npm > 2.1.0
* ruby and compass installed. Install ruby first and then commpass (http://compass-style.org/install/)(http://coolestguidesontheplanet.com/upgrading-ruby-osx/)(http://stackoverflow.com/questions/22544754/failed-to-build-gem-native-extension-installing-compass)
